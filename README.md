\## Giới thiệu

Hệ thống phát hiện hành vi nguy hiểm trong trường học bằng YOLO + LSTM + Django.



\## Pipeline

Camera → YOLO → LSTM → Web → Alert



\## Chức năng

\- Phát hiện bạo lực

\- Phát hiện vũ khí

\- Cảnh báo realtime



\## Cách chạy

pip install -r requirements.txt  

python manage.py runserver
